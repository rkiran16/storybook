# Drop Down

<p>
  Dropdown component is built using Choices.js library. Choices.js is a
  lightweight, configurable select box/text input plugin. Similar to Select2 and
  Selectize but without the jQuery dependency.
</p>

<p>
  For all config options, visit the{" "}
  <a href="https://github.com/Choices-js/Choices">GitHub repo</a>.
</p>

## Setup

<Canvas isColumn>
  <code>
    <div>const element = document.querySelector('.js-choice');</div>
    <div>const choices = new Choices(element);</div>
  </code>
</Canvas>
