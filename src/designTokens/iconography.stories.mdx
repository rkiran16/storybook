# ICON

<p>Icons help communicate meaning, actions, status, or feedback.</p>

## About the icon component

Icons are a simple, easily graspable way to add visual emphasis or interest, signal an action, or indicate a feedback state — all while reducing a user’s cognitive load.

Using a consistent set of icons helps establish a familiar look and feel across a site or service, while using individual icons
consistently establishes a reliable relationship between the image and a specific concept or action.

We use <b>"Material Icons"</b> to populate Icons. Here is the list of Icons <a href="https://fonts.google.com/icons?icon.set=Material+Icons">https://fonts.google.com/icons?icon.set=Material+Icons</a>
